Установка

  1. Установите один из доступных расширений для вашего браузера Tampermonkey или Greasemonkey и Violentmonkey

    • Google Chrome
    • Mozilla Firefox
    • Microsoft Edge или новый Edge
    • Safari
    • Opera

  1. Установка Spaces+

    Вы можете установить этот скрипт через GreasyFork или GitHub

    • GreasyFork: https://greasyfork.org/scripts/397890-spaces
    • GitHub: https://spaces-dev.github.io/spaces-plus.user.js
  • Функции
  • Конфиденциальность
  • Изменения
  • Фоны
  • Звуки

Настройки:

  • Пакетное удаление комментариев
  • Пакетное удаление блогов
  • Пакетное удаление читателей
  • Возможность добавлять пользователей в закладки
  • Кнопка поворота фото/видео в просмотрщике
  • Кнопка ускорения видео
  • Открытые разделы удаленных пользователей
  • Прокрутка страницы справа
  • Скрытие правого меню
  • Кнопка загрузки трека из плеера
  • Переход через внешние ссылки без редиректа
  • Поиск и подтверждение бонусных монет
  • Автоматическое подтверждение кармы
  • Темная тема
  • Точное время онлайн в анкетах
  • Блокировщик рекламы
  • Возможность изменять звук уведомлений
  • Панель друзей онлайн
  • Бесплатные стикеры
  • Возвращает кнопку почту и ленты на свои прежние места
  • Возможность изменять фон сайта
  • Виджет почты

Встроенный возможности сайта:

  • API Отладчик
  • Вход в Beta-песочницу
  • Возмжность добавить полосу загрузки страницы
  • Возможность закрепить аудиоплеер в левом меню

Дополнительно:

  • Редактор cookies

Если вы используете "Импорт и экспорт настроек", то расширение выполнит безопасный запрос на crashmax.ru чтобы получить информацию для валидации JSON во время сохранения его в ваших cookies. Это было сделано для того, чтобы сайт не блокировал доступ к https://jsonformatter.curiousconcept.com/process из-за отсутствующего у него Access-Control-Allow-Origin (MDN Web Docs) заголовка в HTTP запросе.

Все остальные опции содержатся в домене, на котором работает расширение (spaces.ru, spac.me, spcs.me, spaces.im, gdespaces.com, spac1.com и еще spaces-dev.github.io) и никогда не покидает эти домены.
